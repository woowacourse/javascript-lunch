## 어플리케이션 실행 방법!

### 1. 패키지 다운받기

```bash
npm i
```

위 명령어로 필요한 패키지를 다운로드해 주세요!

### 2. 프로젝트 실행

```bash
npm run start
```

위 명령어를 터미널에 입력해주세요!

### 3. 점심 메뉴를 등록해보세요!

```
추가한 음식점은 로컬스토리지에 저장됩니다!
새로고침 시 저장된 데이터를 읽어옵니다!
```
