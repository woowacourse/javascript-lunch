## 프로그램 설명
점심 메뉴 추천 웹사이트

## 기능 요구사항

- [ ]  음식점 목록 페이지 구현
- [ ]  우측 상단의 추가 버튼을 눌러 모달창을 띄운다
- [ ]  모달창에서 새로운 음식점 추가
    - [ ]  카테고리, 거리는 셀렉트 박스를 사용
        - [ ]  카테고리는 "한식", "중식", "일식", "아시안", "양식", "기타" 중 하나를 선택
        - [ ]  거리는  5, 10, 15, 20, 30 중 하나를 선택
    - [ ]  이름/설명/참고 링크는 텍스트 인풋
    - [ ]  카테고리, 이름, 거리 입력 필수
- [ ]  입력값이 잘못되었을 때 사용자에게 알림
