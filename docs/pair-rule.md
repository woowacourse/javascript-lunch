# 👥 페어 규칙

## 💡 **공통**

- **페어 규칙**
  -  전환 시간) **15분** (협의를 통해 유동적으로 변경 가능)
  - 헨드폰 타이머를 바탕으로 시간 철저히 지키기
  - 커밋을 못하더라도 그냥 넘기기
  - 라이브쉐어 사용x -> 맥북 화면공유 기능으로 변경 
  - 페어에게 노트북 양도 (1대의 노트북으로만 작업)
- **커밋 기록**
  - source control 이용 / 기능 단위 
- **커밋 규칙**
  - feat: (새로운 기능)
  - fix: (기능 수정)
  - refactor: (기능 리팩토링)
  - test: (테스트 코드 관련)
  - style: (코드 포맷팅)
  - design: (css 관련)
  - docs: (문서 관련)
  - chore: (자잘부리 -> 파일명 수정)
